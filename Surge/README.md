## ⚙️ 主要功能

- 🔄 智能分流：各国节点智能选择
- 🤖 支持 AIGC 服务（ChatGPT等）
- 📱 电报信息独立策略
- 🍎 苹果服务优化
- Ⓜ️ 微软服务优化
- 📺 流媒体服务支持
- 🛡️ 广告过滤与隐私保护

## 🌐 策略组说明

- 🎯 **节点选择**：主要代理选择
- ✈️ **电报信息**：Telegram专用
- 🤖 **AIGC**：AI服务专用
- Ⓜ️ **微软服务**：Microsoft相关服务
- 🍎 **苹果服务**：Apple相关服务
- 📺 **看看剧**：流媒体服务专用
